# pronunt-frontend-service

Next.js frontend with TypeScript and shadcn/ui for the PRISM dashboard.

## Stack

* Next.js
* TypeScript
* shadcn/ui-ready component setup
* Docker Hardened Images with multi-stage builds
* Kubernetes raw manifests in `k8s/`
* Helm chart in `helm/`
* GitHub Actions workflow in `.github/workflows/`

## Branching

This repository follows trunk-based development with `main` as the long-lived branch.
