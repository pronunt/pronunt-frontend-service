# Commit Conventions

Use Conventional Commits for this repository.

Format:

`type(scope): short summary`

Examples:

* `feat(ui): add PR dashboard layout`
* `fix(table): handle empty state rendering`
* `refactor(components): simplify card composition`
* `docs(readme): update local setup steps`
* `chore(deps): bump Next.js`
* `test(ui): add dashboard component tests`

Recommended types:

* `feat`
* `fix`
* `refactor`
* `docs`
* `chore`
* `test`
* `ci`
* `build`
